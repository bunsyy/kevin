# Project Context: Kevin's Portfolio

## Overview

This project is a personal portfolio website for Kevin, an AI Engineer and Fullstack Developer. The project is built using React, TypeScript, and Vite.

## Purpose

The primary purpose of this website is to showcase Kevin's expertise in AI engineering and fullstack development, highlight his professional experience, and attract both full-time and freelance opportunities.

## Tech Stack

- **Frontend Framework**: React 19
- **Build Tool**: Vite 7
- **Language**: TypeScript
- **Styling**: Vanilla CSS (default template styles currently in place)
- **Package Manager**: Yarn

## Project Structure

- `src/`: Contains the source code.
  - `main.tsx`: Entry point of the application.
  - `App.tsx`: Main application component.
  - `App.css`: Styles for the App component.
  - `index.css`: Global styles.
  - `assets/`: Directory for static assets like images and icons.
- `public/`: Static assets that are copied directly to the build output.
- `index.html`: The main HTML file.
- `package.json`: Project dependencies and scripts.
- `tsconfig.*.json`: TypeScript configuration files.
- `vite.config.ts`: Vite configuration.

## Development Workflow

- **Run Locally**: `npm run dev` or `yarn dev`
- **Build**: `npm run build` or `yarn build`
- **Lint**: `npm run lint` or `yarn lint`
- **Preview Build**: `npm run preview` or `yarn preview`

## User Flow

Currently, the application displays the default Vite + React template home page. The planned user flow will include:

1.  **Home/Hero**: Professional introduction.
2.  **About Me**: Detailed background as an AI engineer and fullstack developer.
3.  **Experience**: Timeline of work history (full-time and freelance).
4.  **Projects**: Showcase of AI and web development projects.
5.  **Skills**: Technical proficiency overview.
6.  **Contact**: Contact information and freelance inquiry options.
